# Plaza

## Setup

Run `RUNME.sh` which will set everything up

OR

- `npm install`
- `pip install -r requirements.txt`

To prevent _Invalid Host Header_ error:

- `echo "DANGEROUSLY_DISABLE_HOST_CHECK=true" > .env.development.local`

## Requirements

1. Flask
2. React
3. python-dotenv
4. Flask-SQLAlchemy
